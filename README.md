# Sandbox

Sandbox 是一个个人技术实验场，汇集了我在不同方向探索和实践的各类项目。每个模块都是独立的“沙盒”，彼此隔离，自由生长。

---

## 🧩 模块概览

| 模块 | 方向 | 描述 | 状态 |
| :--- | :--- | :--- | :--- |
| **sandbox-living** | 物联网 | 智能设备中控：摄像头监控、传感器数据采集、远程控制 | 🟢 实验ing |
| **sandbox-ai** | 人工智能 | 机器学习/深度学习算法实践（规划中） | ⚪ 待启动 |
| **sandbox-vision** | 计算机视觉 | 图像识别、目标检测实验（规划中） | ⚪ 待启动 |
| **sandbox-tools** | 效率工具 | 日常开发小工具集合（规划中） | ⚪ 待启动 |

*更多模块陆续添加中...*

---

## 🎯 项目初衷

> 每个想法都值得一块实验田。

Sandbox 是我的代码游乐场，没有业务压力，没有交付期限，只有对技术的热爱和好奇。
