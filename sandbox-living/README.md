# sandbox-living

## 🎯 实验目标

构建一个轻量级的物联网设备中控系统，实现对家庭/办公环境的设备统一管理。目前专注于三个核心能力：**视觉监控**、**环境感知**和**远程控制**。

---

## 🧩 功能特性

| 功能 | 描述 | 状态 |
| :--- | :--- | :--- |
| **摄像头监控** | 实时查看摄像头画面，支持多路视频流 | 🟢 实验ing |
| **传感器监测** | 采集温度、湿度、光照等环境数据 | 🟡 开发中 |
| **设备控制** | 远程开关灯光、插座等设备 | 🟡 开发中 |
| **数据可视化** | 展示传感器历史数据趋势图 | ⚪ 待启动 |
| **告警通知** | 设备异常时触发告警 | ⚪ 待启动 |

---

## 🛠️ 技术选型

| 分类 | 技术栈 | 说明 |
| :--- | :--- | :--- |
| **核心框架** | Java 17 + Spring Boot 3.4.4 | 基础开发框架 |
| **视频处理** | JavaCV / OpenCV | 摄像头视频流采集 |
| **硬件通信** | 串口通信 / MQTT | 传感器数据读取、设备控制 |
| **数据库** | MySQL + MyBatis-Plus | 数据持久化 |
| **工具库** | Hutool、Lombok、Fastjson2 | 开发效率工具 |
